BOT_TOKEN - токен бота, можно сделать своего, или написать мне.

В коде, для выбора поезда в двух местах нужно ввести XPATH кнопки ВЫБРАТЬ БИЛЕТЫ у нужных поездов(их можно взять на другиx датах)(вставить в лист selectors)
Еще в нужном месте нужно вставить ссылку на страницу выбора билетов, она заметна(driver.get)
